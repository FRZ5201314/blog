年初时用 Python 写的一个图片分辨率 2 次幂批量修正的工具。

贴图、图片压缩分辨率必须是 2 次幂才行。

通过手动修改、一张张去找用格式工厂修改，显然不切实际。

于是下班回家花了两小时边学 python 边写了这个工具。

# 一、工具下载

我把工具放在了阿里云盘：

下载链接：https://www.aliyundrive.com/s/WbtjxrFba3Y

因为阿里云目前不支持压缩包格式分享

文件格式为 jpg，下载后改后缀为 rar 并解压即可

# 二、用法

1. 解压后编辑 textureAmentDic.txt，内容修改为你要批量处理图片的文件夹

2. 运行 textureAment.exe，等待批量修正完成即可。

另外，在当前的版本中，我加入了文件夹选择功能。

删掉 textureAmentDic2.txt 文件，运行 textureAment.exe，就可以通过资源管理器选择需要批量处理的图片的文件夹
