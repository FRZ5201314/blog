# Unity Camera.WorldToScreenPoint 返回值突变的解决办法

在做 UI 同步的时候，必不可少的要用到 Camera.WorldToScreenPoint 来做坐标转换。

然而，又经常会出现 WorldToScreenPoint 返回值 正负突变 的问题。

网上也有小伙伴遇到了同样问题

# 分析原因：

当物体离开相机视野，并且处于相机后方时。

# 解决办法：

判断物体是否在相机的后方，以此来做数值修正。
